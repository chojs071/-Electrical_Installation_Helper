# Electrical Installation Helper (전기설비 도우미)

**Electrical Installation Helper**는 전기기능사 필기 3과목 및 전기(공사)기사 및 전기(공사)산업기사 필기 5과목 공부를 효율적으로 도와주는 보조 도구(프로그램)입니다.

---

## 주요 기능 (Key Features)

- **각 과목 내용에 맞는 KEC규정 알리미**
  - KEC규정에 맞는 전기설비 기준

## 주 목적

- **기존 LLM들은 최신 KEC규정을 알려주지 않아 공부하는 과정중에 어려움이 있습니다.**
  - 예를 들면 LLM모델의 지식 컷오프 이후에 개정된 규정에 관해서는 LLM모델이 환각현상으로 인해 정확하지 않는 정보를 알려주는 문제로 인해 공부하는 과정에서 혼란이 있는 등의 문제점이 있습니다.
  - LLM모델의 지식 컷오프 이전에 개정된 규정에 관한 정보도 올바르지 않거나 타국의 규정을 알려주는 문제점이 있습니다.
- **최신 KEC규정에 맞는 전기설비 기준을 알려주는 사이트를 만들기로 하였습니다.**
